## 1. Getting Started

### 1.1 What is IMS?

The Inventory Management System (IMS) is a web-based application that allows you to:

- **Request items** from the organization's inventory
- **Track your requests** through the approval process
- **View items** currently assigned to you
- **Return items** when no longer needed
- **Manage inventory** (for authorized personnel)

### 1.2 System Requirements

- **Browser:** Google Chrome, Mozilla Firefox, Microsoft Edge (latest versions)
- **Internet Connection:** Required
- **Screen Resolution:** Minimum 1280 x 720 recommended

### 1.3 Accessing the System

The IMS can be accessed through:

1. **Digital System (DS):** Click the "IMS" link from your DS dashboard
2. **Direct URL:** Contact your IT department for the direct access URL

## 2. Logging In

### 2.1 Login via Digital System (Recommended)

1. Log in to the **Digital System (DS)** with your credentials
2. Click on **"IMS"** or **"Inventory Management"** from the menu
3. You will be automatically logged in to IMS

### 2.2 Direct Login

1. Navigate to the IMS login page
2. Enter your **Username** and **Password**
3. Click **"Sign In"**

### 2.3 After Login - DS Landing Page

After logging in from DS, you will see the landing page with two options:

| Option | Description |
|--------|-------------|
| **Personal IMS** | Access your personal dashboard, request items, view your inventory |
| **Wing IMS** | Access wing-level functions (for supervisors only) |

Click on the appropriate option based on your needs.

## 4. Requesting Items (Stock Issuance)

### 4.1 How to Request Items

**Step 1:** Click **"Request Item"** from the menu (or navigate to Stock Issuance Personal)

**Step 2:** Fill in the request details:

| Field | Description | Required |
|-------|-------------|----------|
| Request Type | Personal Expenditure, Returnable, or Individual | Yes |
| Category | Filter items by category (optional) | No |
| Priority | Low, Normal, High, or Critical | Yes |
| Expected Return Date | For returnable items only | Conditional |
| Justification | Reason for the request | No |

**Step 3:** Search and select items:

1. Type in the **search box** to find items
2. Click the **"+"** button to add an item to your request
3. Adjust the **quantity** as needed
4. Repeat for additional items

**Step 4:** Click **"Submit Request"**

### 4.2 Request Types Explained

| Type | Description | Return Required |
|------|-------------|-----------------|
| **Personal Expenditure** | Consumable items (pens, paper, etc.) | No |
| **Returnable** | Items that must be returned (equipment, tools) | Yes |
| **Individual** | Items for personal use with return date | Yes |

### 4.3 Adding Custom Items

If you can't find the item you need:

1. Look for **"Can't find your item? Add a custom item"** link below the search box
2. Click on it to open the custom item form
3. Enter the **item name** and **quantity**
4. Click **"Add Custom Item"**

> **Note:** Custom items go through the same approval process as regular items.

### 4.4 Priority Levels

| Priority | When to Use |
|----------|-------------|
| **Low** | Item needed within 2+ weeks |
| **Normal** | Standard request (1-2 weeks) |
| **High** | Urgent need (within a week) |
| **Critical** | Emergency requirement (immediate) |

## 5. Tracking Your Requests

### 5.1 Viewing Your Requests

1. Click **"My Requests"** from the menu
2. You will see a list of all your requests with their status

### 5.2 Request Status Meanings

| Status | Meaning |
|--------|---------|
| **Pending** | Awaiting approval |
| **Approved** | Approved, waiting for issuance |
| **Processing** | Being prepared for issuance |
| **Issued** | Items have been issued to you |
| **Rejected** | Request was denied (see rejection reason) |
| **Returned** | Request was sent back for modification |
| **Partially Issued** | Some items issued, others pending |

### 5.3 Viewing Request Details

1. Click on any request to see full details
2. View:
   - Items requested
   - Current approval status
   - Approval history
   - Comments from approvers

### 5.4 Editing a Returned Request

If your request is returned for modification:

1. Go to **My Requests**
2. Find the returned request
3. Click **"Edit"**
4. Make the required changes
5. Click **"Resubmit"**

## 6. Returning Items

### 6.1 How to Return Items

1. Click **"Return Item"** from the menu
2. Select the items you want to return from the list
3. Enter the **return quantity**
4. Add any **notes** about the condition
5. Click **"Submit Return"**

### 6.2 Return Process

1. You submit the return request
2. Store keeper verifies the items
3. Items are returned to inventory
4. Your issued items list is updated

## 7. For Wing Supervisors

### 7.1 Wing Dashboard

As a Wing Supervisor, you have access to:

- **Wing Dashboard:** Overview of your wing's inventory and requests
- **Wing Requests:** Requests from your wing members
- **Wing Inventory:** Items assigned to your wing
- **Wing Members:** Members in your wing
- **Approval Dashboard:** Pending approvals from your subordinates

### 7.2 Approving Requests

1. Go to **"My Approvals"** or **"Approval Dashboard"**
2. Review pending requests from your wing members
3. For each request, you can:
   - **Approve:** Send to next level or processing
   - **Reject:** Deny the request (provide reason)
   - **Forward:** Send to another approver

### 7.3 How to Approve

1. Click on a pending request
2. Review the items and justification
3. Click **"Approve"** or **"Reject"**
4. If rejecting, enter the **rejection reason**
5. Confirm your action

### 7.4 Forwarding a Request

If a request is outside your approval authority:

1. Click **"Forward"**
2. Select the appropriate approver
3. Add forwarding notes (optional)
4. Click **"Forward Request"**

## 8. For Store Keepers

### 8.1 Store Keeper Dashboard

As a Store Keeper, you have access to:

- **Forwarded Verifications:** Requests forwarded for your verification
- **Verification History:** Previously processed verifications
- **Wing Inventory:** Inventory in your assigned wing

### 8.2 Processing Verifications

1. Go to **"Forwarded Verifications"**
2. Review each request
3. Verify item availability
4. **Approve** or **Reject** the verification
5. Add notes if needed

### 8.3 Issuing Items

1. Go to **"Stock Issuance Processing"**
2. View approved requests ready for issuance
3. Verify stock availability
4. Click **"Issue"** to complete the issuance
5. System will update inventory automatically

## 9. For Procurement Officers

### 9.1 Procurement Dashboard

As a Procurement Officer, you can:

- Create and manage **Tenders**
- Manage **Vendors**
- Create **Purchase Orders**
- Receive **Deliveries**

### 9.2 Creating a Tender

1. Go to **Dashboard → Contract/Tender**
2. Click **"Create New Tender"**
3. Fill in tender details:
   - Tender Number
   - Title
   - Type (Contract/Annual/Patty Purchase)
   - Opening and Closing dates
4. Add items to the tender
5. Click **"Save"** or **"Publish"**

### 9.3 Managing Vendors

1. Go to **Dashboard → Vendors**
2. View existing vendors or add new ones
3. Assign vendors to tenders

### 9.4 Creating Purchase Orders

1. Go to **Dashboard → Purchase Orders**
2. Click **"Create PO"**
3. Select the tender or enter details manually
4. Add items and quantities
5. Select vendor
6. Click **"Create"**

### 9.5 Receiving Deliveries

1. Go to the Purchase Order
2. Click **"Receive Delivery"**
3. Enter received quantities
4. Add inspection notes
5. Click **"Confirm Receipt"**

## 11. Frequently Asked Questions

### Q1: How long does it take for my request to be approved?
**A:** Approval time depends on your organization's workflow. Typically:
- Low priority: 3-5 business days
- Normal: 2-3 business days
- High: 1-2 business days
- Critical: Same day (if approvers are available)

### Q2: Can I cancel a submitted request?
**A:** You cannot cancel a request after submission. Contact your supervisor if you need to cancel.

### Q3: Why was my request rejected?
**A:** Check the rejection reason in your request details. Common reasons:
- Insufficient justification
- Item not available
- Budget constraints
- Duplicate request

### Q4: How do I know when my items are ready for pickup?
**A:** You will receive a notification when items are issued. Check your notifications or "My Issued Items" page.

### Q5: Can I request items for someone else?
**A:** Only Wing Supervisors can request items on behalf of their wing members.

### Q6: What if the item I need is not in the catalog?
**A:** Use the "Add Custom Item" feature to request items not in the catalog.

### Q7: How do I change my password?
**A:** Password changes are managed through the Digital System (DS). Contact IT support if needed.

### Q8: Who approves my requests?
**A:** Requests follow your organization's approval hierarchy:
1. Wing Supervisor (first level)
2. Admin/Department Head (if required)
3. Final approval (for high-value items)

## 12. Troubleshooting

### Issue: Can't log in

**Solutions:**
1. Verify your username and password
2. Clear browser cache and cookies
3. Try a different browser
4. Contact IT support

### Issue: Page not loading

**Solutions:**
1. Check your internet connection
2. Refresh the page (F5)
3. Clear browser cache
4. Try again after a few minutes

### Issue: Request not submitting

**Solutions:**
1. Ensure all required fields are filled
2. Check that you have at least one item added
3. Verify your session hasn't expired (log in again)

### Issue: Can't find an item

**Solutions:**
1. Try different search terms
2. Check if the item is in a different category
3. Use the "Add Custom Item" feature
4. Contact the inventory manager

### Issue: Notifications not appearing

**Solutions:**
1. Refresh the page
2. Check your notification settings
3. Clear browser cache
