## Summary

The hierarchical inventory system provides:

1. **Location Awareness**: Separate inventories for admin and each wing
2. **Intelligent Routing**: Automatically deducts from correct location
3. **Forwarding Capability**: Moves requests to admin if wing insufficient
4. **Complete Audit Trail**: Every movement logged in stock_transfer_log
5. **Transaction Safety**: All operations atomic with rollback support
6. **Status Tracking**: request_inventory_source tracks location and approval flow

This enables healthcare facilities to:
- âœ… Manage wing-level inventory independently
- âœ… Maintain central admin warehouse for replenishment
- âœ… Forward requests when wings lack stock
